FAQ/Suporte

Perguntas Frequentes

Como faço para criar uma conta?

Para a criação de uma conta em nosso site são necessários os seguintes dados: CPF, telefone, E-mail. Com esses dados em mãos é necessário somente seguir o passo a passo que a plataforma irá repassar.

A plataforma se responsabiliza pela contratação do serviço?

Não, nós não nos responsabilizamos pelos serviços. Nossa apenas plataforma apresenta os serviços de nossos prestadores.

Qual o valor da taxa para que eu possa disponibilizar meus serviços?

A taxa para disponibilizar nossos serviços é de R$150 por mês.

Como faço caso eu esqueça minha senha?

Para recuperar a senha de acesso é necessário acessar nossa página de recuperação de senha. Passo a passo: Página Inicial -> Esqueci minha senha -> Digitar o e-mail cadastrado -> Chegará o link via e-mail e gerar uma nova senha.
